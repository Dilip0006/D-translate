D Lingo – Live Translator App

D Lingo is a prototype application that converts voice to text and translates it into another language.

Features

- Voice input
- Text translation
- Voice output
- Multi language support

How to Run

1. Install Node.js
2. Create React project

npx create-react-app dlingo

3. Go to project folder

cd dlingo

4. Start the app

npm start

Future Features

- Live calling translation
- Secure login
- Cloud database
- Android mobile app
